# Docker & Kubernetes: The Practical Guide

Course notes and practice projects from the Udemy course  
**Docker & Kubernetes: The Practical Guide** by Maximilian Schwarzmüller (Academind).

🔗 [Course link](https://www.udemy.com/course/docker-kubernetes-the-practical-guide/)

---

## What this course covers

- Introduction to Docker and containers — what they are and why use them
- Installing and using Docker on Windows, macOS and Linux
- Creating and managing images and containers
- Core Docker CLI commands
- Persisting data with volumes
- Container networking — communication with the outside world and between containers
- Single and multi-container projects with Docker Compose
- Deploying Docker applications to the cloud (AWS)
- Introduction to Kubernetes — clusters, pods, deployments and services
- Deploying and managing apps with Kubernetes

## Tech stack used in examples

- Node.js
- Python
- PHP / Laravel

## Structure

Each folder corresponds to a section of the course.
